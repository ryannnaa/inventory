# Rental Inventory Management - Frontend

React + Vite frontend application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Ensure backend is running on port 3001

3. Start development server:
```bash
npm run dev
```

App runs on http://localhost:5173

## Features

- 🔍 Inventory Search with real-time availability
- 📋 Current Rentals tracking
- 🚚 Delivery Schedule
- ↩️ Upcoming Returns
- ⚠️ Low Availability Alerts
